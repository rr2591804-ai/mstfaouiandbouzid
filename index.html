مشروع تصميم صفحة ويب

ما هي لغة HTML ؟

HTML هي لغة تستخدم لإنشاء صفحات الويب. تستعمل لكتابة العناوين والفقرات والصور والروابط.

ما هي لغة CSS ؟

CSS تستعمل لتجميل الموقع وتغيير الألوان والخطوط والتصميم.

ما هي JavaScript ؟

JavaScript تستعمل لجعل الموقع تفاعلي مثل الأزرار والرسائل.
